清掃スケジュールの使い方

従業員配布用・清掃スケジュール画面の操作手順（PDF保存・印刷用）

1. この画面でできること

  • 自分の予定（スケジュール）の確認 … 日別・週間・月間で表示
  • 「入れない時間」の登録（ブロック・クローズ） … 日別表示で空き時間をタップして登録
  • 案件の店舗情報・カルテの閲覧 … 予定をタップすると下部にカルテが表示されます

注意：予定（案件）の追加・変更・削除はできません。閲覧と「入れない時間」の登録のみ可能です。

2. スケジュール画面の開き方

  1. トップ（玄関）で「清掃」を選ぶ
  2. 清掃用の画面でログイン／入室する
  3. 画面内の「予定」をタップする → スケジュール画面が開きます

3. 表示の切り替え（日別・週間・月間）

表示内容日付の移動
日別1日分のタイムライン（時間ごと）前日 / 今日 / 翌日
週間月曜〜日曜の7日分をカード表示前週 / 今週 / 翌週
月間当月のカレンダー前月 / 今月 / 翌月（日付をクリックするとその日を指定）

ヘッダーにある「表示」のボタン（日・週・月のアイコン）で切り替えられます。

4. 日別表示での操作

4.1 タイムラインの見方

縦に時間（0:00〜24:00）、各時間に次のいずれかが表示されます。

  • 案件 … 割り当てられた清掃予定（店舗名・時間など）
  • 🔒 クローズ … 自分で登録した「入れない時間」
  • 空き（タップでブロック作成） … 予定のない時間。ここをタップすると「入れない時間」を登録できます

4.2 「入れない時間」の登録（ブロック作成）

  1. 日別表示で、空きスロットの「空き（タップでブロック作成）」をタップ（クリックまたは右クリック）
  2. モーダルが開くので、次の項目を入力します。
項目必須説明
開始○入れない時間の開始日時（15分刻み）
終了○入れない時間の終了日時。終了は開始より後である必要があります
理由○睡眠 / 移動 / 私用 / その他から選択
メモ任意例）病院、など

「作成」を押すと登録されます。既存の予定や他のクローズと時間が重なると登録できません。その場合は開始・終了時刻を変更して再度お試しください。

4.3 案件をタップして店舗情報・カルテを見る

  1. タイムライン上の案件（予定）のカードをタップ
  2. 画面下部にカルテが表示されます
  3. 左側：店舗名・日付・時間・種別、および店舗の連絡先・キーボックス解錠番号など
  4. 右側：店舗の詳細カルテ（閲覧のみ。編集はできません）
  5. カルテを閉じる：カルテ上部の「×」ボタンをタップ。高さは上部のヘッダーをドラッグして変更できます

5. 週間・月間表示での操作

週間表示

7日分が横並びで表示されます。各日のカード内の案件をタップすると、日別と同様にカルテが開きます。週間表示では「入れない時間」の登録はできません。登録する場合は日別表示に切り替えて行ってください。

月間表示

当月のカレンダーが表示されます。日付をクリックすると、その日を基準にした表示に変わります。ブロックの登録やカルテの表示は、日別または週間に切り替えて行ってください。

6. よくあるメッセージと対処

表示される内容対処
認証が必要ですエントランスに戻るリンクから再度ログインしてください
清掃員情報の取得に失敗しました管理者に連絡してください。表示の「エントランスに戻る」で戻れます
409 Conflict（重複のため登録できません）「入れない時間」が既存の予定や他のクローズと重なっています。開始・終了時刻を変えて再度登録してください
ブロックの作成に失敗しました通信エラーなどの可能性があります。時刻を確認して再度お試しください。続く場合は管理者に連絡してください

7. 操作一覧（早見）

やりたいこと操作
自分の予定を見る「予定」からスケジュール画面を開く。日別・週・月で表示を切り替え可能
日付を変える前日/翌日・前週/翌週・前月/翌月ボタンで移動
入れない時間を登録する日別表示で「空き」をタップ → 開始・終了・理由を入力 → 作成
案件の店舗情報・カルテを見る案件カードをタップ → 下部にカルテ表示。×で閉じる

お問い合わせ：画面の不具合や操作で困ったことがあれば、管理者までご連絡ください。
