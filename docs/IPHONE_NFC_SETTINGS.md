# iPhoneのNFC設定ガイド

## iPhoneのNFC機能について

### NFC機能の有効化

iPhoneでは、**NFC機能は常に有効**になっています。特別な設定は不要です。

- **iPhone 7以降**: NFC機能が利用可能
- **iPhone XS以降**: バックグラウンドNFCタグ読み取りが可能（画面ロック中でも読み取り可能）

### NFC設定の確認方法

1. **設定アプリ**を開く
2. **「コントロールセンター」**を選択
3. **「NFCリーダー」**がコントロールセンターに追加されているか確認（オプション）

**注意**: iPhoneのNFC機能は通常、設定でON/OFFを切り替える必要はありません。常に有効です。

## NFCタグの読み取り方法

### 標準的な動作

1. **iPhoneをNFCタグに近づける**（数cm以内）
2. **自動的にSafariが開く**（URLが書き込まれている場合）
3. **追加アプリ不要**

### ブラウザの選択

- **デフォルト**: Safariで開く（iOS標準）
- **Chromeで開く場合**: Chromeをデフォルトブラウザに設定している場合、Chromeで開く可能性があります

### Chromeで開く場合の対処法

1. **設定アプリ**を開く
2. **「Safari」**を選択
3. **「デフォルトのWebブラウザ」**を確認
4. **「Safari」**を選択（Chromeではなく）

または、NFCタグを読み取る際に、**Safariで開く**を選択してください。

## トラブルシューティング

### NFCタグが読み取れない

1. **iPhoneをNFCタグに近づける**（数cm以内、背面の上部）
2. **画面をロック解除**する（iPhone XS以降は不要）
3. **NFCタグが正しく書き込まれているか確認**
4. **NFCタグが破損していないか確認**

### Chromeで開いてエラーになる

**原因**: Chromeがデフォルトブラウザに設定されている、またはChromeで開くことを選択した

**解決方法**:
1. NFCタグを読み取る際に、**「Safariで開く」**を選択
2. または、iPhoneの設定で**Safariをデフォルトブラウザ**に設定

### URLが開かない

1. **インターネット接続を確認**
2. **URLが正しく書き込まれているか確認**
3. **Safariの設定を確認**

## 推奨設定

### デフォルトブラウザをSafariに設定

1. **設定アプリ**を開く
2. **「Safari」**を選択
3. **「デフォルトのWebブラウザ」**で**「Safari」**を選択

これにより、NFCタグを読み取った際に、常にSafariで開くようになります。

## 注意事項

- iPhoneのNFC機能は、**画面ロック中でも動作します**（iPhone XS以降）
- NFCタグにURLが書き込まれている場合、**自動的にブラウザが開きます**
- **追加アプリは不要**です（iOS標準機能）
