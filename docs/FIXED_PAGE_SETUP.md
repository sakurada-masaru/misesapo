# 固定ページのセットアップ確認

## 📋 現在の状況

- 固定ページを作成済み
- ショートコードを追加する必要がある

---

## 🔍 確認事項

### 1. 固定ページにショートコードが追加されているか

固定ページのElementorエディターで、以下のショートコードが追加されているか確認してください：

1. `[misesapo_hero]` - ヒーローセクション
2. `[misesapo_contact_box]` - お問い合わせボタンセクション
3. `[misesapo_problem]` - お悩みセクション

---

## 📝 ショートコードの追加方法

### ステップ1: Elementorエディターを開く

1. WordPress管理画面 → **固定ページ** → **すべてのページ**
2. 作成した固定ページを開く
3. **「Elementorで編集」** をクリック

### ステップ2: ショートコードを追加

1. 左側のウィジェットパネルから **「HTML」** または **「Code」** ウィジェットをドラッグ&ドロップ
2. ウィジェット内に以下のショートコードを入力：

```
[misesapo_hero]
```

3. 同様に、もう1つのHTMLウィジェットを追加：

```
[misesapo_contact_box]
```

4. もう1つのHTMLウィジェットを追加：

```
[misesapo_problem]
```

### ステップ3: ページを保存

1. **「更新」** または **「公開」** をクリック
2. **「ページを表示」** をクリックして、フロントエンドで確認

---

## 🎯 表示されるべき内容

### 1. ヒーローセクション (`[misesapo_hero]`)
- フルスクリーンの画像スライダー（3枚の画像がフェードイン・フェードアウト）
- マスク画像
- 化粧ヘッダー（ロゴとナビゲーション）
- スクロールヒント

### 2. お問い合わせボタンセクション (`[misesapo_contact_box]`)
- 「まずはご気軽にご相談してみませんか？」の見出し
- キャッチフレーズ
- 電話番号ボタン（070-3332-3939）
- Webでのお申し込みボタン

### 3. お悩みセクション (`[misesapo_problem]`)
- 「こんなお悩みありませんか？」の見出し
- お悩みリスト（4項目）
- イラスト画像

---

## 🚨 表示されない場合の確認

### ショートコードが文字として表示される場合

- ショートコードが正しく登録されていない可能性
- `functions.php` でショートコードが登録されているか確認

### 何も表示されない場合

1. **テンプレートファイルが存在するか確認**
   - `/lightning-child/templates/hero-section.php`
   - `/lightning-child/templates/contact-box-section.php`
   - `/lightning-child/templates/problem-section.php`

2. **functions.php でショートコードが登録されているか確認**
   - `add_shortcode('misesapo_hero', ...)`
   - `add_shortcode('misesapo_contact_box', ...)`
   - `add_shortcode('misesapo_problem', ...)`

3. **画像ファイルがアップロードされているか確認**
   - `/lightning-child/assets/images/images-admin/`
   - `/lightning-child/assets/images/images-material/`
