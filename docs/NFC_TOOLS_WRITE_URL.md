# NFC ToolsでURLを書き込む方法

## 正しい書き込み手順

### ステップ1: NFC Toolsアプリを開く

1. **NFC Tools**アプリを開く
2. **「書き込み」**タブを選択（下部のタブバー）

### ステップ2: URLを選択

1. **「レコードを追加」**をタップ
2. **「URL」**を選択（「Webサイト」や「リンク」ではなく、**「URL」**を選択）

### ステップ3: URLを入力

1. **URLフィールド**に、以下の形式でURLを入力：
   ```
   https://misesapo.co.jp/staff/dashboard?nfc_tag_id=TAG_001
   ```
   
   **重要**: 
   - `http://`ではなく`https://`を使用
   - 完全なURLを入力（省略しない）
   - パラメータ（`?nfc_tag_id=TAG_001`）も含める

2. **「書き込み」**ボタンをタップ

### ステップ4: NFCタグに書き込む

1. iPhoneをNFCタグに近づける（数cm以内、背面の上部）
2. 書き込みが完了するまで待つ
3. 「書き込み完了」のメッセージが表示される

## よくある間違い

### ❌ 間違い1: 「Webサイト」を選択

- **「Webサイト」**を選択すると、Google検索が開かれる可能性があります
- **必ず「URL」を選択**してください

### ❌ 間違い2: URLが不完全

- `misesapo.co.jp/staff/dashboard`（`https://`がない）
- `staff/dashboard?nfc_tag_id=TAG_001`（ドメインがない）

### ❌ 間違い3: テキストとして書き込む

- **「テキスト」**として書き込むと、URLとして認識されません
- **必ず「URL」として書き込む**必要があります

## 書き込み内容の確認方法

### 方法1: NFC Toolsで読み取る

1. NFC Toolsアプリを開く
2. **「読み取り」**タブを選択
3. NFCタグにiPhoneを近づける
4. 書き込まれた内容を確認

**確認ポイント**:
- **「URL / URI」**として表示されているか（「Webサイト」や「テキスト」ではない）
- URLが正しく表示されているか（`https://misesapo.co.jp/staff/dashboard?nfc_tag_id=TAG_001`）
- **複数のレコードが追加されていないか**（URLレコードのみであることを確認）

### 重要: 複数のレコードに注意

NFC Toolsでは、複数のレコードを追加できます。**URLレコード以外のレコード（Webサイト、テキストなど）が追加されていると、iPhoneが別のレコードを読み取ってGoogleが開かれる可能性があります。**

**確認方法**:
- 書き込み画面で、URLレコード以外のレコードが表示されていないか確認
- 不要なレコードがある場合は、削除してから書き込む

### 方法2: iPhoneで直接読み取る

1. iPhoneをNFCタグに近づける
2. Safariが自動的に開くか確認
3. 正しいURLが開かれているか確認

## トラブルシューティング

### Googleが開かれる場合

**原因**: 
- 「URL」ではなく「Webサイト」や「テキスト」として書き込まれている可能性
- URLレコード以外のレコード（Webサイトなど）が追加されている可能性
- 複数のレコードが書き込まれており、iPhoneがURLレコード以外を読み取っている

**解決方法**:
1. **書き込み画面で確認**: URLレコード以外のレコードが表示されていないか確認
2. **不要なレコードを削除**: URLレコード以外のレコードがある場合は削除
3. **NFC Toolsで読み取って確認**: タグを読み取って、書き込まれた内容を確認
4. **再度書き込む**: URLレコードのみが表示されている状態で、再度書き込む

### Safariが開かない場合

**原因**:
- URLが正しく書き込まれていない
- NFCタグが破損している

**解決方法**:
1. NFC Toolsでタグを読み取って確認
2. 必要に応じて、新しいNFCタグを使用

## 正しい書き込み例

### ダッシュボードを開く場合

```
https://misesapo.co.jp/staff/dashboard?nfc_tag_id=TAG_001
```

### マイページを開く場合

```
https://misesapo.co.jp/staff/mypage?nfc_tag_id=TAG_001
```

**重要**: 
- `https://`から始まる完全なURL
- ドメイン名（`misesapo.co.jp`）を含む
- パラメータ（`?nfc_tag_id=TAG_001`）も含む
